Minecraft: Java & Bedrock Edition

Создавайте, исследуйте и выживайте снова и снова. Minecraft: Java Edition и Bedrock Edition теперь входят в комплект версии для Windows. Версия Minecraft: Java & Bedrock Edition дает доступ сразу к обеим играм и содержит универсальный лаунчер для них, позволяющий легко переключаться с одного выпуска на другой. Теперь вы можете играть с любым пользователем Minecraft на другой платформе, просто переключившись на используемую им версию. В два раза больше удовольствия от добычи ресурсов, создания и совместной игры с другими пользователями Minecraft!
